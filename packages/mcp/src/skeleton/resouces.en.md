# Skeleton Component Tools

## Description
MCP tools for the Skeleton component. Used to manage loading state.

## Tools List

### getLoading
Get the loading state.

**Parameters**: None

**Return Value**: 
```json
{
  "loading": true
}
```

### setLoading
Set the loading state.

**Parameters**:
- `loading` (boolean): Whether loading

**Return Value**: `success`
