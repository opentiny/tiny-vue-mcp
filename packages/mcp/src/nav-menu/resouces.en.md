# NavMenu Component Tools

## Description
MCP tools for the NavMenu component. Used to manage navigation menu state.

## Tools List

### getActiveKey
Get the currently active menu key.

**Parameters**: None

**Return Value**: 
```json
{
  "activeKey": "key-string"
}
```

### setActiveKey
Set the active menu key.

**Parameters**:
- `key` (string): Menu key

**Return Value**: `success`
