# Watermark Component Tools

## Description
MCP tools for the Watermark component. Used to manage watermark text.

## Tools List

### getText
Get the watermark text.

**Parameters**: None

**Return Value**: 
```json
{
  "text": "watermark-text"
}
```

### setText
Set the watermark text.

**Parameters**:
- `text` (string): Watermark text

**Return Value**: `success`
