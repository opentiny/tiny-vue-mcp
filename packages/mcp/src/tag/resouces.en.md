# Tag Component Tools

## Description
MCP tools for the Tag component. Used to manage tag types and states.

## Tools List

### getType
Get the tag type.

**Parameters**: None

**Return Value**: 
```json
{
  "type": "success|info|warning|danger"
}
```

### setType
Set the tag type.

**Parameters**:
- `type` (string): Tag type

**Return Value**: `success`

### close
Close the tag.

**Parameters**: None

**Return Value**: `success`
