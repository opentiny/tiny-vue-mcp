# Link Component Tools

## Description
MCP tools for the Link component. Used to manage link addresses.

## Tools List

### getHref
Get the link address.

**Parameters**: None

**Return Value**: 
```json
{
  "href": "url-string"
}
```

### setHref
Set the link address.

**Parameters**:
- `href` (string): Link address

**Return Value**: `success`
