# Empty Component Tools

## Description
MCP tools for the Empty component. Used to manage empty state descriptions.

## Tools List

### getDescription
Get the empty state description.

**Parameters**: None

**Return Value**: 
```json
{
  "description": "string"
}
```

### setDescription
Set the empty state description.

**Parameters**:
- `description` (string): Description text

**Return Value**: `success`
