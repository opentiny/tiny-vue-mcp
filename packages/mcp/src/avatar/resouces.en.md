# Avatar Component Tools

## Description
MCP tools for the Avatar component. Used to get and set avatar source.

## Tools List

### getSrc
Get the avatar source.

**Parameters**: None

**Return Value**: 
```json
{
  "src": "url-string"
}
```

### setSrc
Set the avatar source.

**Parameters**:
- `src` (string): Avatar URL

**Return Value**: `success`
