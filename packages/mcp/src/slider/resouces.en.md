# Slider Component Tools

## Description
MCP tools for the Slider component. Used to get and set slider values.

## Tools List

### getValue
Get the current slider value.

**Parameters**: None

**Return Value**: 
```json
{
  "value": number
}
```

### setValue
Set the slider value.

**Parameters**:
- `value` (number): Slider value to set

**Return Value**: `success`

### reset
Reset the slider to minimum value.

**Parameters**: None

**Return Value**: `success`
