# TreeMenu Component Tools

## Description
MCP tools for the TreeMenu component. Used to manage tree menu nodes.

## Tools List

### getSelectedNode
Get the currently selected node.

**Parameters**: None

**Return Value**: 
```json
{
  "selectedNode": "node-id"
}
```

### setSelectedNode
Set the selected node.

**Parameters**:
- `nodeId` (string): Node ID

**Return Value**: `success`
