# Cascader Component Tools

## Description
MCP tools for the Cascader (Cascading Select) component. Used to manage the selected values of cascading selectors.

## Tools List

### getValue
Get the currently selected value.

**Parameters**: None

**Return Value**: 
```json
{
  "value": []
}
```

### setValue
Set the value of the cascader.

**Parameters**:
- `value` (any[]): Array of values to set

**Return Value**: `success`

### clear
Clear the cascader value.

**Parameters**: None

**Return Value**: `success`
