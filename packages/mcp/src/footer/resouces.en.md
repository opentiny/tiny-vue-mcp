# Footer Component Tools

## Description
MCP tools for the Footer component. Used to manage page footer content.

## Tools List

### getContent
Get the page footer content.

**Parameters**: None

**Return Value**: 
```json
{
  "content": "string"
}
```

### setContent
Set the page footer content.

**Parameters**:
- `content` (string): Footer content

**Return Value**: `success`
