# Divider Component Tools

## Description
MCP tools for the Divider component. Used to get and set divider content.

## Tools List

### getContent
Get the divider content.

**Parameters**: None

**Return Value**: 
```json
{
  "content": "string"
}
```

### setContent
Set the divider content.

**Parameters**:
- `content` (string): Divider content text

**Return Value**: `success`
