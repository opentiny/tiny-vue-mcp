# Badge Component Tools

## Description
MCP tools for the Badge component. Used to get and set badge values.

## Tools List

### getValue
Get the current badge value.

**Parameters**: None

**Return Value**: 
```json
{
  "value": "value"
}
```

### setValue
Set the badge value.

**Parameters**:
- `value` (any): Badge value (can be a number or string)

**Return Value**: `success`
