# Image Component Tools

## Description
MCP tools for the Image component. Used to get and set image source.

## Tools List

### getSrc
Get the image source.

**Parameters**: None

**Return Value**: 
```json
{
  "src": "string"
}
```

### setSrc
Set the image source.

**Parameters**:
- `src` (string): Image URL

**Return Value**: `success`
