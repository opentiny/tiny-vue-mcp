# Layout Component Tools

## Description
MCP tools for the Layout component. Used to manage layout modes.

## Tools List

### getMode
Get the layout mode.

**Parameters**: None

**Return Value**: 
```json
{
  "mode": "vertical|horizontal"
}
```

### setMode
Set the layout mode.

**Parameters**:
- `mode` (string): Layout mode

**Return Value**: `success`
