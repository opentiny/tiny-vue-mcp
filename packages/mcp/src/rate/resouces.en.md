# Rate Component Tools

## Description
MCP tools for the Rate (Rating) component. Used to get and set rating values.

## Tools List

### getValue
Get the current rating value.

**Parameters**: None

**Return Value**: 
```json
{
  "value": number
}
```

### setValue
Set the rating value.

**Parameters**:
- `value` (number): Rating value between 0-5

**Return Value**: `success`

### clear
Clear the rating.

**Parameters**: None

**Return Value**: `success`
