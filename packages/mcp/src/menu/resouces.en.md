# Menu Component Tools

## Description
MCP tools for the Menu component. Used to get and set active menu items.

## Tools List

### getActiveIndex
Get the index of the currently active menu item.

**Parameters**: None

**Return Value**: 
```json
{
  "activeIndex": 0
}
```

### setActiveIndex
Set the active menu item.

**Parameters**:
- `index` (string): Menu item index

**Return Value**: `success`
