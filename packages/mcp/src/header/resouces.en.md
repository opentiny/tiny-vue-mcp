# Header Component Tools

## Description
MCP tools for the Header component. Used to manage page header title.

## Tools List

### getTitle
Get the page header title.

**Parameters**: None

**Return Value**: 
```json
{
  "title": "string"
}
```

### setTitle
Set the page header title.

**Parameters**:
- `title` (string): Title text

**Return Value**: `success`
