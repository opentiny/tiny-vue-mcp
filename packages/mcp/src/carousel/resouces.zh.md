# Carousel 组件工具

## 描述
Carousel（走马灯/轮播）组件的 MCP 工具集合。用于获取、设置和管理轮播的当前索引。

## 工具列表

### getCurrentIndex
获取当前轮播索引。

**参数**: 无

**返回值**: 
```json
{
  "currentIndex": number
}
```

### setCurrentIndex
设置轮播到指定索引。

**参数**:
- `index` (number): 要设置的索引

**返回值**: `success`

### next
切换到下一张。

**参数**: 无

**返回值**: `success`

### prev
切换到上一张。

**参数**: 无

**返回值**: `success`
