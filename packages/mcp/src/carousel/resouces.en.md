# Carousel Component Tools

## Description
MCP tools for the Carousel component. Used to get, set, and manage the current index of carousel slides.

## Tools List

### getCurrentIndex
Get the current carousel index.

**Parameters**: None

**Return Value**: 
```json
{
  "currentIndex": number
}
```

### setCurrentIndex
Set carousel to a specified index.

**Parameters**:
- `index` (number): Index to set

**Return Value**: `success`

### next
Move to the next slide.

**Parameters**: None

**Return Value**: `success`

### prev
Move to the previous slide.

**Parameters**: None

**Return Value**: `success`
