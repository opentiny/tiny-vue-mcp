# Spin Component Tools

## Description
MCP tools for the Spin component. Used to manage spinning loading state.

## Tools List

### getSpinning
Get the spinning loading state.

**Parameters**: None

**Return Value**: 
```json
{
  "spinning": false
}
```

### setSpinning
Set the spinning loading state.

**Parameters**:
- `spinning` (boolean): Whether to spin

**Return Value**: `success`
