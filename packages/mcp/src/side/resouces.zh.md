# Side 组件工具

## 描述
Side（侧边栏）组件的 MCP 工具集合。用于管理侧边栏折叠状态。

## 工具列表

### getCollapsed
获取侧边栏是否折叠。

**参数**: 无

**返回值**: 
```json
{
  "collapsed": false
}
```

### setCollapsed
设置侧边栏折叠状态。

**参数**:
- `collapsed` (boolean): 是否折叠

**返回值**: `success`

### toggleCollapsed
切换侧边栏折叠状态。

**参数**: 无

**返回值**: `success`
