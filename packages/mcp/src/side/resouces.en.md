# Side Component Tools

## Description
MCP tools for the Side (Sidebar) component. Used to manage sidebar collapsed state.

## Tools List

### getCollapsed
Get whether the sidebar is collapsed.

**Parameters**: None

**Return Value**: 
```json
{
  "collapsed": false
}
```

### setCollapsed
Set the sidebar collapsed state.

**Parameters**:
- `collapsed` (boolean): Whether to collapse

**Return Value**: `success`

### toggleCollapsed
Toggle the sidebar collapsed state.

**Parameters**: None

**Return Value**: `success`
