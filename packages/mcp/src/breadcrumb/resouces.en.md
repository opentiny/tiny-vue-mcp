# Breadcrumb Component Tools

## Description
MCP tools for the Breadcrumb component. Used to manage breadcrumb items.

## Tools List

### getItems
Get the list of breadcrumb items.

**Parameters**: None

**Return Value**: 
```json
{
  "items": []
}
```

### setItems
Set breadcrumb items.

**Parameters**:
- `items` (array): Array of breadcrumb items

**Return Value**: `success`
