# Calendar Component Tools

## Description
MCP tools for the Calendar component. Used to get, set, and reset date values.

## Tools List

### getValue
Get the currently selected date.

**Parameters**: None

**Return Value**: 
```json
{
  "value": "date-string"
}
```

### setValue
Set the calendar date.

**Parameters**:
- `date` (string): Date string

**Return Value**: `success`

### reset
Reset the calendar to the current date.

**Parameters**: None

**Return Value**: `success`
