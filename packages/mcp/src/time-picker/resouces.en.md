# TimePicker Component Tools

## Description
MCP tools for the TimePicker component. Used to get and set time values.

## Tools List

### getValue
Get the currently selected time.

**Parameters**: None

**Return Value**: 
```json
{
  "value": "HH:MM:SS"
}
```

### setValue
Set the time picker value.

**Parameters**:
- `value` (string): Time value in format "HH:MM:SS"

**Return Value**: `success`

### clear
Clear the time selection.

**Parameters**: None

**Return Value**: `success`
