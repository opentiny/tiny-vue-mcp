# TimePicker 组件工具

## 描述
TimePicker（时间选择器）组件的 MCP 工具集合。用于获取和设置时间值。

## 工具列表

### getValue
获取当前选择的时间。

**参数**: 无

**返回值**: 
```json
{
  "value": "HH:MM:SS"
}
```

### setValue
设置时间选择器的值。

**参数**:
- `value` (string): 时间值，格式如 "14:30:00"

**返回值**: `success`

### clear
清空时间选择。

**参数**: 无

**返回值**: `success`
