# Switch Component Tools

## Description
MCP tools for the Switch component. Used to get and set switch state.

## Tools List

### getValue
Get the current switch state.

**Parameters**: None

**Return Value**: 
```json
{
  "value": boolean
}
```

### setValue
Set the switch state.

**Parameters**:
- `value` (boolean): true for on, false for off

**Return Value**: `success`

### toggle
Toggle the switch state.

**Parameters**: None

**Return Value**: `success`
