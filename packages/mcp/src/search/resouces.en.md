# Search Component Tools

## Description
MCP tools for the Search component. Used to get, set, and clear search values.

## Tools List

### getValue
Get the current search value.

**Parameters**: None

**Return Value**: 
```json
{
  "value": "string"
}
```

### setValue
Set the search value.

**Parameters**:
- `value` (string): Search value

**Return Value**: `success`

### clear
Clear the search value.

**Parameters**: None

**Return Value**: `success`
