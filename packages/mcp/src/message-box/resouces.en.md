# MessageBox Component Tools

## Description
MCP tools for the MessageBox component. Used to display and manage message boxes.

## Tools List

### show
Show message box.

**Parameters**:
- `options` (object): Message box options (title, message, type)

**Return Value**: `success`

### confirm
Confirm action.

**Parameters**: None

**Return Value**: `confirmed`

### cancel
Cancel action.

**Parameters**: None

**Return Value**: `cancelled`
